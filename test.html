Dashboard Test Page

Open the browser console (F12) to see debugging information when you:

  1. Enter edit mode (click the wrench icon)
  2. Create a new card (click the + button on any card)
  3. Add icons to the new card (click the + inside the new card)

You should see:

  • New cards start small (120px height)
  • Icon spacing remains consistent (16px gap)
  • Cards expand as you add more icons
